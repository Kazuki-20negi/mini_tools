# 小プログラム
主に学習目的で作った簡単なプログラムや、ちょっとした便利ツール、ジョークツールの詰め合わせです。

## path_display_system
ディレクトリツリーをCLIで表示するものです。<br>
既存のものがあるようですが、再帰等の学習のため作成しました。

## password_maker
randomモジュールではなく、secretsモジュールを使用してより安全にパスワードを生成します。<br>
ArgumentParserを使用して、コマンドラインからの実行時に直接桁数を渡して動作します。<br>
-u または --unicode と入力すると、ユニコード文字を含むパスワードを生成します。
使用場面は限られますが、理論上、より安全です。

## スターリンソート
SNSで一時期話題になった、ソートされていないデータを粛清することによってソートを実現するプログラムです。
