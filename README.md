# 小プログラム
主に学習目的で作った簡単なプログラムや、ちょっとした便利ツール、ジョークツールの詰め合わせである。

## path_display_system
ディレクトリツリーをCLIで表示するものである。<br>
既存のものがあるようだが、再帰等の学習のため作成した。

## password_maker
randomモジュールではなく、secretsモジュールを使用してより安全にパスワードを生成する。<br>
ArgumentParserを使用して、コマンドラインからの実行時に直接桁数を渡して動作する。<br>
-u または --unicode と入力すると、ユニコード文字を含むパスワードを生成できる。使用場面は限られるものの、理論上、より安全。

## stalin_sort
SNSで一時期話題になった、ソートされていないデータを粛清することによってソートを実現するプログラムである。

## LINE_talklog_search
ラインのトークログを解析する。<br>
参加ログ、退出ログ、累計参加人数、特定ユーザの発言文字数を調べることができる。これらはLINEの純正の機能ではできないものである。

## num_in_text
とある用途で、ログ中の数字をすべて足す処理をする必要があったため作成した。<br>
複数行入力に対応しており、wwhileループにより繰り返し処理を行える
